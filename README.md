# DID Contract Indexer POC

DID Contract Indexer Proof of Concept - A TypeScript-based indexer for DID smart contracts.

## Features

- TypeScript support
- ESLint for code linting
- Build and development scripts

## Prerequisites

- Node.js >= 18.x
- npm or yarn or pnpm

## Installation

```bash
npm install
```

## Usage

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Run

```bash
npm start
```

### Linting

```bash
npm run lint
npm run lint:fix
```
